# MITRE ATT&CK for Web Applications

## Tactic: Discovery (TA0007)
*   **Technique T1046 (Network Service Discovery)**: Scan for open ports and services.
*   **Technique T1087 (Account Discovery)**: Enumerate valid usernames.
*   **Technique T1426 (System Information Discovery)**: Identify OS, web server versions.

## Tactic: Initial Access (TA0001)
*   **Technique T1190 (Exploit Public-Facing Application)**: SQLi, RCE, XSS.
*   **Technique T1078 (Valid Accounts)**: Default credentials, weak passwords.

## Tactic: Credential Access (TA0006)
*   **Technique T1110 (Brute Force)**: Password spraying, credential stuffing.
*   **Technique T1555 (Credentials from Password Stores)**: Finding config files with keys.

## Execution Guidance
*   **Mapping**: Map your actions to these TTPs (Tactics, Techniques, and Procedures).
*   **Adversary Emulation**: Think like an attacker. What is the path of least resistance?
