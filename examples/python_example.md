# Python Code Execution Example

The agent can now execute Python scripts generated by the LLM!

## How It Works

1. **LLM generates Python code** in a ```python block
2. **Agent detects** the code block
3. **Saves** to temporary file
4. **Executes** with `python3`
5. **Captures output** and sends back to LLM
6. **Output Analyzer** summarizes if needed

## Example Usage

### Agent Prompt:
```
Analyze the nmap scan results and extract only open HTTP/HTTPS ports
```

### LLM Response:
````
I'll write a Python script to parse the nmap output:

```python
import json

# Sample nmap data (replace with actual)
nmap_output = """
PORT     STATE SERVICE
80/tcp   open  http
443/tcp  open  https
8080/tcp open  http-proxy
"""

# Parse and extract HTTP ports
http_ports = []
for line in nmap_output.strip().split('\n')[2:]:  # Skip header
    parts = line.split()
    if len(parts) >= 3:
        port = parts[0].split('/')[0]
        service = parts[2]
        if 'http' in service.lower():
            http_ports.append(port)

# Clean output
print(f"Found {len(http_ports)} HTTP/HTTPS ports:")
print(json.dumps(http_ports, indent=2))
```
````

### Output Sent Back to LLM:
```
Python Script Output (Summary):
Found 3 HTTP/HTTPS ports: 80, 443, 8080
```

## Best Practices

### ✅ DO:
- Print **concise summaries** instead of full data
- Use **JSON** for structured output
- Include **error handling** (try/except)
- Count items instead of listing all

### ❌ DON'T:
- Print massive datasets
- Use verbose debugging output
- Forget error handling
- Print unnecessary information

## Use Cases

1. **Data Processing**: Parse complex tool outputs (JSON, XML, CSV)
2. **Custom Exploits**: Generate payloads or attack scripts
3. **Analysis**: Calculate statistics from scan results
4. **Automation**: Chain multiple operations
5. **Format Conversion**: Transform data between formats
